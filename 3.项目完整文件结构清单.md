## 项目架构特点

1. **Monorepo 结构**: 使用 PNPM workspace 管理多个应用和包
2. **微前端架构**: 分离教师端、学生端、管理控制台
3. **微服务架构**: 后端按业务领域分离为多个服务
4. **组件化开发**: 独立的 UI 组件库
5. **TypeScript 支持**: 全面的类型安全
6. **现代工具链**: Vite、Turbo、ESLint、Prettier 等
